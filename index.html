Canvas Lines

  • 1. By default drawing lines on the canvas produces blurry lines: 

    
            ctx1.beginPath();
            ctx1.moveTo(10, 0);
            ctx1.lineTo(10, 90);
            ctx1.moveTo(20, 0);
            ctx1.lineTo(70, 50);
            ctx1.stroke();
            ctx1.closePath();

  • 2. Experiment with drawing evenly spaced lines: 

    
            ctx2.lineWidth = parseInt(lineWidth);
            ctx2.beginPath();
            for (var i = 0; i < 100; i += parseInt(spacing) + 1) {
              ctx2.moveTo(i, 0);
              ctx2.lineTo(i, 100);
            }
            ctx2.stroke();
            ctx2.closePath();
            ctx2.lineWidth = 1;


  • 3. Offset the canvas by -0.5 pixels: 

    
            ctx3.clearRect(0, 0, 100, 100);
            ctx3.translate(-0.5, -0.5);
            ctx3.lineWidth = parseInt(lineWidth);

            // Line 1
            ctx3.beginPath();
            ctx3.moveTo(0, 0);
            ctx3.lineTo(0, 40);
            ctx3.stroke();

            // Line 2
            ctx3.beginPath();
            ctx3.moveTo(25, 0);
            ctx3.lineTo(60, 40);
            ctx3.stroke();

            // Stripes
            for (var i = 0; i < 100; i += parseInt(spacing) + 1) {
              ctx3.beginPath();
              ctx3.moveTo(i, 50);
              ctx3.lineTo(i, 100);
              ctx3.stroke();
            }
            ctx3.lineWidth = 1;
            ctx3.resetTransform();
